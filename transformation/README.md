xslt
====

Repository til prototypen til XSLT af HTML til ScriplerXML/XHTML (SXML/SXHTML).

* Example1: spec in "HTML - ScriplerXML - #1.jpg", using "procedural" (flat) XSLT
* Example2: (X)HTML -> (X)HTML, using apply-templates
* Example3: new input from David (product owner): divide into frontmatter, contents and backmatter
* Example4: new input from David (product owner): more flexible structure, cf. "ScriplerXHTML - transformations.jpg"

Tip: for at se resultatet af XSL-transformationen i Firefox, tryk CTRL+A og vælg "View Selection Source".
